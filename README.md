# 🛒 Tienda Online de Bicicletas

Proyecto web desarrollado como práctica de desarrollo full stack.

## 🔧 Tecnologías utilizadas
- PHP
- HTML
- CSS
- JavaScript
- MySQL

## ⚙️ Funcionalidades
- Sistema de login y registro de usuarios
- Carrito de compra
- Gestión de productos
- Panel de administración

## 🧪 Ejecución en local
Este proyecto requiere un servidor local (XAMPP, WAMP, etc.) para funcionar correctamente.

## 📂 Base de datos
Incluida en el archivo: tienda_online.sql

## 📌 Nota
El proyecto está subido como demostración de código, por lo que no se ejecuta directamente en GitHub Pages.
